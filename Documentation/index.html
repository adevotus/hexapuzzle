1Open project in Unity Editor (use 2018.4.9f or higher version for project to work perfectly)

Click on Open button and find your Hexa Puzzle - Unity Template folder, which you downloaded after purchase successful.

2How to change Icon and Product Name

1 Drag any icon you have designed for the game into project folder (Recommend "Images" folder inside project).

2 Click on "File/Build Settings/".


3 When "Build Setting" popup show, click "Player Setting".


4 Drag the icon you prepare earlier into "Default icon" to replace the game icon.


5 Input the name you want for the game in "Product Name" to replace the name of the game.


3 How to change Package Name Very Important

1 Click on "Other Setting" in "Player Setting" .

2 Input the Package Name you want for the game in "Package Name" to replace the game Package Name. .

4 How to use Ads Controller

1 To active Admob, click on Asset/Google Mobile Ads/Settings.

2 Fill your Admob AppID into field like the image below.

3 To active Unity Ads, follow step by step in this link

4 Choose InitObject in Game Scene.

5 In there, you can fill all the Admob ID you have prepared in the Input Field we create for you.

5 We prepared for you 3 interstitial events, to increase you just need to increse the number of elements, after do that you can see a new ads event created.

6 You can see a new ads event create:

- AdType for which ads provider your want to use, we already prepared Unity Ads and Admob.

- AdPlacement for where you want to show ads.

Example: The event in picture, we use Admob interstitial when Game Over, that mean if player lose the game the interstitial ads of Admob will show.

5 How to reskin in project

- To reskin, you could choose the Image object and drag your new image you prepare to change.

Example: I chose the play button object, and drag my picture I have prepared in to Image.
